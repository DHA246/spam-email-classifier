# Spam Email Classifier

A machine learning project that classifies email/SMS messages as Spam or Ham (Not Spam).

## Features
- Detects spam messages using NLP
- Text preprocessing and feature extraction
- Machine learning based classification
- Simple and efficient prediction system

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy
- TensorFlow
- Jupyter Notebook

## Dataset
The model is trained on a labeled spam dataset.

## How to Run
1. Install dependencies:
pip install -r requirements.txt

2. Run the application:
python app.py

## Project Goal
To build an intelligent text classification system for spam detection.
